Overview
--------
Audit Claude Code, Codex CLI, and OpenCode for autonomous multi-step task
execution with file output. Results determine E4-07b architectural approach:
SDK-based (current plan) vs. agent-orchestrated (preferred if viable).

The bar is: can at least one agent run a multi-step task to completion,
reading structured input files and writing structured output files, without
human intervention?

See findings.md for the full test protocol and results matrix.
